This is the frontend application for the Sweet Shop Management System built using React, TypeScript, Vite, and Tailwind CSS.

## Features
- User registration and login
- JWT-based authentication
- Sweet listing dashboard
- Search sweets by name and category
- Purchase sweets
- Admin panel for restocking and management
- Dark mode UI

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Axios
- React Router

```bash
npm install
npm run dev
